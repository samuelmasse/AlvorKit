# AlvorKit.FreeType.Native

FreeType binaries for win-x64, linux-x64 and osx-arm64

The package version is the upstream [FreeType](https://gitlab.freedesktop.org/freetype/freetype) release the binaries were built from (tag `VER-x-y-z`). Built with the internal zlib and without bzip2/png/harfbuzz/brotli, so the binaries have no dependencies.

| RID | Binary |
| --- | --- |
| win-x64 | `freetype.dll` |
| linux-x64 | `libfreetype.so` |
| osx-arm64 | `libfreetype.dylib` |
